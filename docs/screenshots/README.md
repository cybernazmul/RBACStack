# Screenshots

Place the following screenshot files in this folder so they appear in the root README.md:

| Filename | Page | How to capture |
|---|---|---|
| `login.png` | Login page | http://localhost:5173/login |
| `dashboard.png` | Dashboard (full page) | http://localhost:5173/dashboard |
| `users.png` | Users page | http://localhost:5173/users |
| `roles.png` | Roles page | http://localhost:5173/roles |
| `permissions.png` | Permissions page | http://localhost:5173/permissions |
| `audit-logs.png` | Audit Logs page | http://localhost:5173/audit-logs |
| `sidebar-collapsed.png` | Sidebar in collapsed mode with flyout open | Collapse sidebar → click User Management |

**Tip:** Use browser DevTools → Ctrl+Shift+P → "Capture full size screenshot" for a clean full-page capture.
